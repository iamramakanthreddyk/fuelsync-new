# Expense Workflow - Quick Reference

## The ₹1,700 Pending Expense Explained

### What's Happening?
```
Status: ⏳ Awaiting Review
Amount: ₹1,700
Count: 1 pending

Why?
→ Someone (likely staff) entered this expense
→ It needs manager approval before it counts in reports
→ You can approve it manually OR bulk-approve it
```

### How to Fix It - Choose One:

#### Option 1: Manual Approval (Safest)
1. Go to **Expenses** page
2. Look for **"Pending Approvals (1)"** section at top
3. See the ₹1,700 expense details
4. Click ✅ **Approve** button
5. Done! Now shows in "Approved Total"

#### Option 2: Safe Bulk Approve (Recommended)
1. Go to **Expenses** page  
2. In **"Pending Approvals"** section
3. Click **"Safe Auto-Approve"** button
4. This auto-approves it (₹1,700 is under ₹10k limit)
5. Done!

#### Option 3: Approve All (Use with Caution)
1. Go to **Expenses** page
2. In **"Pending Approvals"** section  
3. Click **"Approve All"** button
4. Confirm when asked
5. All pending expenses approved (⚠️ no going back!)

## Key Concepts

| Concept | Meaning |
|---------|---------|
| **Pending** | Waiting for manager to review & approve |
| **Approved Total** | Expenses you've approved (counts in reports) |
| **Safe Auto-Approve** | Smart filtering - only low-risk items |
| **Who can approve?** | Only Managers, Owners, Super Admin |

---

## Why Is It Pending?

```
When EMPLOYEES create expenses:
  → Status: PENDING
  → Reason: Extra safeguard against fraud/mistakes
  → Who approves: Manager/Owner
  → How long: Usually within 24 hours

When MANAGERS create expenses:
  → Status: AUTO_APPROVED (instant!)
  → Reason: Managers are accountable
  → No waiting needed
  → Counts immediately
```

## Pro Tips

✅ Check "Pending Approvals" section daily
✅ Use "Safe Auto-Approve" for routine items
✅ Manually review high-value expenses (> ₹10,000)
✅ Expenses are logged for audit trail

❌ Don't ignore pending expenses - they delay financial reports
❌ Don't approve without reviewing
❌ Don't worry - you can see who entered each expense

## FAQ

**Q: Can I undo an approval?**
A: No, approvals are final. Be careful when bulk-approving.

**Q: Where did this expense come from?**
A: Click on it to see who entered it and when.

**Q: What if someone entered wrong amount?**
A: Reject it, they can resubmit with correct amount.

**Q: How often should I approve?**
A: Daily or as-needed. Pending expenses aren't counted in reports.
